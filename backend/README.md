# Запуск бэкенда без Docker

Этот каталог содержит исходный код API на FastAPI. Ниже описан способ локального запуска без контейнеров.

## Установка зависимостей

Из корня репозитория создайте виртуальное окружение и установите пакеты:

```bash
python -m venv venv
source venv/bin/activate
pip install -r backend/requirements.txt
```

## Запуск приложения

Находясь в корне проекта, выполните команду:

```bash
python -m uvicorn backend.main:app --reload
```

Сервер стартует на порту `8000`. При необходимости его можно изменить, задав переменную окружения `BACKEND_PORT` или отредактировав файл `.env`.
